# SynBioSuite Documentation

SynBioSuite is a browser-based synthetic biology design and analysis workspace with:

- a React/Vite single-page app in `frontend/`
- a Flask API in `backend/`
- Terraform-based Microsoft identity configuration in `iac/`

This documentation is organized for two audiences:

- **Users**: how the app is entered and used today
- **Developers**: how the codebase is structured and what is currently incomplete

## Current-state note

These docs describe what is visible in the current repository, not an idealized future architecture. Features that are unclear or only partially implemented are marked explicitly, including placeholders where intent is not obvious from code.
