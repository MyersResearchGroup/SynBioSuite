# SynBioSuite MkDocs starter

This folder contains a MkDocs-ready documentation site for the current repository state of SynBioSuite.

## Run locally

```bash
pip install -r requirements-docs.txt
mkdocs serve
```

## Build static site

```bash
mkdocs build
```

MkDocs expects a top-level `mkdocs.yml` file and a `docs/` directory containing Markdown pages.
